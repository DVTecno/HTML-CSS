Mi primer Style

Aquí vamos aplicar estilos

En este ejercicio, crearemos un archivo HTML para establecer la estructura de la página.
Luego, vincularemos un archivo CSS para darle estilo a nuestros elementos.
Asignaremos un color de fondo atractivo y centraremos los títulos H1 y H2 para destacarlos.
También ajustaremos la fuente de los encabezados y párrafos para lograr una apariencia más
personalizada y finalmente, alinearemos los párrafos a la izquierda para una mejor lectura.

Una vez terminado, abriremos el archivo "index.html" en nuestro navegador para ver los cambios aplicados.
Nuestra página mostrará los títulos y el texto con las características que hemos definido previamente.
Con estos pasos, habremos logrado crear una página web simple pero atractiva con HTML y CSS,
demostrando la importancia del diseño en la presentación de contenido en la web.
